About API

The Squash API is responsible for carrying out all information traffic from the login information of a user even the posts that he makes, currently the API is developed in NodeJS and Html having its database in ClearDB(MySql), it is sent and received data in JSON format only.

Security.

Squash values the security of the user and their data, so the AES encryption standard was adopted for common data such as name, username, bio, and for passwords the Hash encryption was adopted.
